# Kmoe Download

Kmoe Download 是一个 Chrome 扩展，用于在已适配的漫画详情页中批量选择章节，并下载 `MOBI` / `EPUB` 文件。

这个项目主要用于个人学习和浏览器扩展开发研究，不是通用下载器，也不提供任何绕过网站限制的能力。

## 功能

- 批量选择章节
- 支持 `MOBI` 和 `EPUB`
- 支持下载队列、并发控制和失败重试
- 支持 aria2 下载和直接下载模式
- 记录近期下载历史
- 历史记录支持查看、导出、导入和清空

## 安装

1. 克隆仓库：

```bash
git clone https://github.com/Fyhax1ki/kmoe-download.git
```

2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启右上角的“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择本项目根目录

## 使用方法

1. 打开已适配的漫画详情页
2. 等待页面内容加载完成
3. 点击页面上的 `Kmoe-Download`
4. 选择文件格式、章节和下载参数
5. 点击“开始下载”
6. 通过扩展弹窗查看或管理下载历史

默认使用 aria2 下载模式。使用前需要在扩展弹窗中打开“aria2 配置”填写 JSON-RPC 信息；如遇到 aria2 环境不可用，可以切换为直接下载。

## 设置

扩展弹窗中可以调整：

- 并发下载数
- 下载间隔
- 最大重试次数
- 下载方式
- aria2 RPC 地址、Token、下载目录和连接参数

设置会自动保存，并用于后续下载任务。

本项目没有构建步骤，修改后重新加载 Chrome 扩展即可生效。

## 说明与限制

- 仅适用于已经适配的页面
- 不保证对所有页面结构长期有效
- 不会绕过登录、额度、权限或访问限制
- 下载行为会受到浏览器、网站响应和网络环境影响
- 请控制下载频率，避免给目标服务造成额外压力
